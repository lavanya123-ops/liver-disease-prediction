# Liver Disease Prediction using Machine Learning

## Project Overview
This project predicts whether a person is likely to have liver disease based on medical parameters using machine learning techniques. The model analyzes patient data such as age, gender, alkaline phosphatase (Alkphos), aspartate aminotransferase (SGOT), total proteins, albumin (ALB), bilirubin levels, and the albumin-to-globulin (A/G) ratio to make predictions.

The goal of this project is to demonstrate how machine learning can assist in the early detection of liver disease and support healthcare decision-making.

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Machine Learning model training
- Prediction of liver disease
- Model evaluation and accuracy measurement

## Dataset
The dataset contains medical attributes related to liver function such as:

- Age
- Gender
- Total Bilirubin
- Direct Bilirubin
- Alkaline Phosphotase (Alkphos)
- Aspartate Aminotransferase (SGOT)
- Total Proteins
- Albumin (ALB)
- Albumin and Globulin Ratio (A/G Ratio)
- Result (Liver disease prediction)
